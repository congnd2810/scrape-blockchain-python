Tổng quan về Fountain Protocol

Fountain Protocol là một nền tảng cross-chain lending đầu tiên được cung cấp bởi Oasis. Giao thức này được xem là một công cụ quản lý vốn đạt hiệu quả cao, cho phép người dùng trải nghiệm nhiều tính năng với đa dạng các tài sản DeFi. 

Tận dụng lợi thế về chi phí thấp và tốc độ giao dịch nhanh chóng của mạng lưới Oasis, Fountain Protocol thiết lập một nền tảng đã đa doanh thu, đồng thời giúp người sử dụng có thể kiếm được các khoản lợi nhuận hấp dẫn.

Mới đây, để chúc mừng cho sự kiện ra mắt bản Testnet của giao thức, dự án đã thu hút được đông đảo sự chú ý của người dùng thông qua việc tổ chức sự kiện Airdrop với phần thưởng vô cùng hấp dẫn.

Hướng dẫn tham gia Testnet Fountain Protocol

Bước 1: Truy cập link dưới đây để tham gia Testnet của Fountain Protocol:

Fountain Protocol Testnet

Bước 2: Thêm mạng Oasis Testnet vào Metamask

Đầu tiên, tại giao diện chính bạn hãy bấm Launch app.

Tiếp theo hãy nhấp vào Emerald Testnet => Cửa sổ MetaMask xuất hiện, bấm Approve => Bấm Switch Network là xong. 

Bước 3: Nhận token Testnet.

Để tham gia Fountain Protocol Testnet, bạn cần một ít token thử nghiệm như ROSE, ETH, USDT, USDC. 

Bạn chỉ cần bấm vào Faucet => Chọn token muốn nhận => Nhập địa chỉ ví => Xác thực => Bấm Request là xong.

Bước 4: Thử nghiệm tính năng Supply

Deposit token

Nhấp vào Market => Chọn Supply => Chọn token muốn deposit => Điền số lượng => Bấm Supply.

Tiếp tục bấm Confirm khi cửa sổ Metamask xuất hiện.

Lưu ý: Nếu đây là lần đầu tiên bạn deposit token này, bạn sẽ cần thực hiện giao dịch hai lần liên tiếp. Lần thứ nhất là để chấp thuận Fountain smart contract rút token từ ví của bạn và lần thứ hai sẽ tiến hành deposit token.

Withdraw token

Tại giao diện chính màn hình sau khi thực hiện deposit, bạn chỉ cần kéo xuống phía dưới.

Bấm Withdraw => Nhập số lượng token => Tiếp tục chọn Withdraw => Confirm MetaMask.

Bước 5: Thử nghiệm tính năng Borrow

Borrow token

Trước khi tiến hành vay, người dùng được yêu cầu ký quỹ một lượng token để làm tài sản thế chấp. 

Tại mục Supply, bạn chọn token muốn ký quỹ => Nhập số lượng => Bật tab Use as Collateral => Bấm Supply => Bấm Confirm MetaMask.

Tiếp theo chọn Borrow => Nhập số lượng token => Chọn Borrow => Bấm Confirm MetaMask.

Repay token

Tại giao diện chính màn hình sau khi thực hiện Borrow => Bạn kéo xuống dưới => Chọn Repay => Nhập số lượng token => Bấm Repay thêm lần nữa => Confirm MetaMask là xong.

Bước 6: Trải nghiệm tính năng Rewards

Claim token

Để kiếm được nhiều rewards, người dùng cần stake token FTP vào nền tảng Fountain Protocol. Trước hết, bạn hãy chọn mục Rewards => Chọn Claim => Tiếp tục bấm claim FTP => Confirm MetaMask.

Stake token

Tại mục Rewards => Bạn kéo xuống chọn Stake => Nhập số lượng token => Bấm stake => Confirm MetaMask.

Kết nối Yuzu DEX 

Bạn cũng có thể deposit token từ Yuzu DEX Liquidity Provider để kiếm thêm phần thưởng cho mình.

Tại mục Rewards => Hãy kéo xuống Yuzu DEX Rewards => Bấm deposit.

Tiếp theo tại mục Add Liquidity => Nhập số lượng token FTP (sau đó hệ thống sẽ tự đồng điền lượng token ROSE có giá trị USD tương ứng) => Bấm Deposit => Confirm MetaMask.

Sau đó tại mục Deposit FTP-ROSE LP Tokens, bạn nhập số lượng LP => Bấm deposit => Confirm MetaMask. 

Thực hiện xong các bước trên là bạn đã thử nghiệm Testnet Fountain Protocol và tham gia thành công sự kiện Airdrop của dự án.
