Polygon Token Swap là một tính năng mới trong Ví Polygon cho phép bạn swap các loại token khác nhau. Được hỗ trợ bởi api 0x, Polygon luôn tìm kiếm mức giá tốt nhất trên thị trường để người dùng tận dụng tối đa mỗi giao dịch swap.
